|| EN

Installing Docker
Docker is an open source platform for developing, shipping and running applications.

It allows to run any application regardless of the operating system.

Important!
. If you don't know Docker, don't worry, we just use it for running Airflow
. If you can't install Docker or don't want to, you can still follow the prep course
. If you already have Airflow running on your computer, you can skip this part

Steps

Pretty straightforward,

1. Go to https://docs.docker.com/get-docker/
2. Select your operating system
3. Follow the instructions, you shouldn't get any trouble
4. Once Docker is installed, you can install Docker Compose
5. Go to https://docs.docker.com/compose/install/
6. Well done!

|||| Instalando Astronomer CLI

|| EN

Installing the Astronomer CLI
Astronomer's open source CLI is the easiest and fastest way to run Apache Airflow on your machine

Running on top of Docker, with two simple commands you get Airflow running

Important!
If you can't install the Astronomer CLI or don't want to, you can still follow the prep course
If you already have Airflow running on your computer, you can skip this part

Steps
Like Docker, it's rather straightforward,

1. Go to https://www.astronomer.io/docs/cloud/stable/develop/cli-quickstart
2. If you are on MacOS -> Homebrew, if Linux -> cURL and if Windows there is a special guide for you.
3. Follow the instructions, until the step 4 included.

You're ready!
